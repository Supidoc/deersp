AIC3X_ASDIntCtrl_TypeDef *  pASDIntfCtrl 
)

Reads the ASD interface control and stores it in a given struct.

Reads the ASD interface control from the ASD interface control register converts the bit values to the corresponding struct member values and stores them in a given struct.

Parameters
haic3xPointer to a AIC3X_HandleTypeDef structure that contains the configuration information for the specified AIC3X.
pASDIntfCtrlPointer to a AIC3X_ASDIntCtrl_TypeDef structure that will be populated with the ASD interface control data.
Returns
Hal Status
Warning
The size of the sent data is not calculated with sizeof() but set with a constant value of 3u because the registers have a length of 24 bits.
